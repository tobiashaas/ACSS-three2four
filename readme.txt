Migrates Automatic.css v2/v3 data to v4 format in Bricks Builder sites.

== Description ==
Go to Tools → ACSS3 to ACSS4 and click "Start Migration".

Prerequisites:
- ACSS4 must be installed and active
- Bricks Builder must be installed and active
- Create a database backup before running
